# App API

Microservice written in .NET 8

## Pre-requisite

- [Nix](https://nixos.org/)
- [direnv](https://direnv.net/)
- [Docker](https://www.docker.com/)

## Setup

Initialize the repository

```bash
pls setup
pls setup:dotnet
```

## Development

Start the development environment

```bash
pls start
```
